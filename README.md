# Upbit Signal Watch

업비트 공개 시세 API로 조건에 맞는 코인을 검색하고, 저장한 조건을 주기적으로 감시해 브라우저 알림을 띄우는 정적 웹앱입니다.

## 실행

이미 로컬 서버가 실행 중이면 아래 주소로 접속합니다.

```text
http://localhost:5178/
```

직접 다시 실행하려면 이 폴더에서 실행합니다.

```powershell
node local-server.js
```

같은 와이파이에 연결된 안드로이드 기기에서 보려면 PC의 내부 IP를 확인한 뒤 아래처럼 접속합니다.

```text
http://PC내부IP:5178/
```

## 기능

- KRW, BTC, USDT, 전체 마켓 필터
- 코인명, 영문명, 심볼 검색
- 24시간 변동률, 거래대금, 현재가 조건 검색
- 1시간봉 기준 RSI 조건 검색
- 조건 저장 및 주기 감시
- 브라우저 알림, 진동, 화면 토스트 알림
- 검색 결과 CSV 내보내기

## 알림 참고

이 앱은 서버 없이 브라우저에서 동작하므로 앱이 열려 있을 때 조건을 감시합니다. 진짜 백그라운드 푸시 알림을 원하면 HTTPS 서버, 백엔드 작업자, 푸시 서비스 구성이 추가로 필요합니다.

모바일 브라우저 알림은 브라우저와 접속 방식에 따라 제한될 수 있습니다. Android Chrome은 보통 HTTPS 또는 localhost에서 알림 권한을 허용합니다.
